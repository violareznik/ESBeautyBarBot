# ES Beauty Bar BUSINESS BOT

Готовый Telegram-бот для салона красоты.

## Как загрузить
1. Распакуй архив.
2. В GitHub нажми Add file → Upload files.
3. Загрузи ВСЕ файлы и папки из архива.
4. Проверь, что есть папка media с картинками.
5. Проверь, что workflow лежит здесь:
   .github/workflows/main.yml
6. В Secrets добавь:
   BOT_TOKEN
   ADMIN_ID
7. Actions → Run ES Beauty Bot → Run workflow.

Важно: main.yml в корне добавлен только для удобства. Рабочий файл находится в .github/workflows/main.yml.
